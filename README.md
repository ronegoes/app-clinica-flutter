# 💰 App Clinica Flutter

Aplicativo completo de cadastro e agendamento desenvolvido em Flutter.

---

## 🚀 Funcionalidades

- Cadastro de medicos, agenmdamentos e procedimentos 
- Categorias (odontologia, etc)
- Gráfico de gastos por atendimento 📊
- Cálculo automático de saldo


---

## 🛠️ Tecnologias

- Flutter
- Firebase
- fl_chart

---
